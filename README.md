# Simple-servomotor
制作一个简易伺服电机的原因
网上带有绝对位置反馈的伺服电机价格都非常昂贵。对于初学FOC的小伙伴来说，并不需要昂贵的电机，而且先从有感的FOC开始学习更容易上手。
于是制作一个 带绝对值反馈的简易伺服电机的想法就产生了。
